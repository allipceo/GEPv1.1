# GEP_041_Home아코디언재설계

**작업일시:** 2026.02.23
**작업자:** 고팀장 (Claude Code)

## 작업 내용

GEP_038 확정 시나리오 기준 Home.jsx 전면 재설계 + WrongReview.jsx 과목 선택 화면 추가.
12개 버튼 평면 나열 방식을 3개 아코디언으로 압축하고, 틀린문제 진입 시 과목별 현황을 먼저 표시.

## 수정 파일

| 파일 | 변경 내용 |
|------|----------|
| `src/pages/Home.jsx` | 12버튼 평면 목록 → 3개 아코디언 (기본 접힘, 대과목 정답률 표시) |
| `src/pages/WrongReview.jsx` | 과목 선택 화면 추가 + 세부과목별 틀린 수 표시 + 틀린 횟수 많은 순 정렬 |

## 변경 전/후

### Home.jsx — 아코디언 구조

**Before:**
```
법령        보험업법 [90문제] [72%]
           상법     [90문제] [-]
           세제재무 [90문제] [-]
           위험관리 [90문제] [-]
손보1부     보증보험 ...
손보2부     재보험   ...
(12개 버튼 항상 노출 → 스크롤 필요)
```

**After:**
```
법령 ▶  72%          ← 클릭 시 펼침
손보1부 ▶  -
손보2부 ▶  -
(3개 아코디언, 기본 접힘 → 스크롤 없음)

[법령 펼친 상태]
법령 ▶  72%
  보험업법  90문제  72%
  상법      90문제  -
  세제재무  90문제  -
  위험관리  90문제  -
```

### WrongReview.jsx — 과목 선택 화면

**Before:**
- 틀린문제 버튼 → 바로 문제 시작 (전체 과목 혼합)
- 최신 틀린 순 정렬

**After:**
```
[과목 선택 화면]
법령     36개
  보험업법 12개  상법 8개  세제재무 10개  위험관리 6개

손보1부   0개  (비활성)
손보2부   0개  (비활성)

전체 36문제
```
- 과목 선택 → 해당 과목 틀린문제만 표시
- 틀린 횟수 많은 순 정렬 (5회 → 4회 → 3회)
- 문제 헤더: "법령 1/36" + "3회 틀림"
- 첫 문제 이전 버튼 → "과목 선택" (선택 화면으로 복귀)
