## examStore.js 변경 내용

`selectFilteredQuestions` 한 줄 변경:

```js
// 변경 전
if (q.round !== state.selectedRound) return false;

// 변경 후
if (state.selectedRound !== '전체' && q.round !== state.selectedRound) return false;
```

`selectedRound === '전체'` 시 round 조건을 완전히 건너뜀 → 전체 회차 반환.

## Question.jsx 변경 내용

1. **filteredQuestions 필터** (examStore와 동일 수정, 없으면 question 페이지 빈 화면):
```js
// 변경 전
if (q.round !== selectedRound) return false

// 변경 후
if (selectedRound !== '전체' && q.round !== selectedRound) return false
```

2. **headerLabel 수정** (전체회 → 전체 로 표시):
```js
// 변경 전
const headerLabel = selectedSubSubject
  ? `${selectedRound}회 ${selectedSubSubject}`
  : `${selectedRound}회 ${selectedSubject}`

// 변경 후
const roundLabel  = selectedRound === '전체' ? '전체' : `${selectedRound}회`
const headerLabel = selectedSubSubject
  ? `${roundLabel} ${selectedSubSubject}`
  : `${roundLabel} ${selectedSubject}`
```
