## 시나리오 개요

**대상 사용자:** 레벨 4 서비스 가입자  
**진입 경로:** 홈 화면 → "모의고사" 메뉴  
**시험 구조:** 실제 시험과 100% 동일
- 1교시: 법령 40문제 (40분)
- 휴식: 15분 (선택적)
- 2교시: 손보1부 40 + 손보2부 40 = 80문제 (80분)

**핵심 원칙:**
- 시작하면 3세트 완주 기본 전제
- 타이머는 절대 시간 기반 (중간 이탈해도 계속 흐름)
- 타임아웃 시 미응답 0점 처리
- 중간 포기 시 경고 + 0점 처리 + 성적 저장
- 재응시 무제한
