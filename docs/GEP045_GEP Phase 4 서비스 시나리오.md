**GEP Phase 4 서비스 시나리오 초안**

**S1. 진입** 홈 화면에 레벨 3 전용 "OX 학습" 버튼 노출. 탭하면 과목 선택 화면으로 이동. 과목은 기존 3과목(손보1부/손보2부/법령) 동일 구조.

**S2. 문제 화면**

* 중앙에 진위형 문장 1개 표시 (`statement_display`)  
* 하단에 O / X 버튼 2개  
* 우측 상단 또는 하단에 출처 표시: `23회 손보2부 23번` (작은 텍스트)  
* 랜덤 출제, `generation_status=OK` 레코드만 사용  
* 진행 카운터 표시: `12/50` 형식

**S3. 정답/오답 피드백**

* 정답: 초록 효과 → 0.8초 후 다음 문제 자동 이동  
* 오답: 빨간 효과 → 정답(O 또는 X) 표시 후 다음 버튼으로 이동  
* 오답 시 해설은 **미제공** (Phase 4 범위 외, 추후 고려)

**S4. 틀린 문제 관리**

* 오답 기록은 서버(Supabase) 저장  
* 학습 종료 후 "틀린 문제만 다시 풀기" 버튼 제공  
* 틀린 문제 모아풀기는 동일한 S2\~S3 플로우 재사용

**S5. 이어하기**

* 과목별 풀이 진행률 로컬 \+ 서버 저장  
* 홈 재진입 시 "이어서 풀기" 옵션 제공

**GEP Phase 4 서비스 시나리오 최종 확정**

**S1. 진입 — 홈 → OX 학습(레벨3) → 과목 선택 → 순차 출제 시작**

**S2. 문제 화면**

* **진위형 문장 표시, O/X 버튼**  
* **출처: `23회 손보2부 23번`**  
* **상단: `Round 2 | 1/150 (누적 151)`**  
* **하단: 이전 / 다음 버튼**

**S3. 피드백**

* **정답: 초록 효과 → 다음 활성화**  
* **오답: 빨간 효과 \+ 정답 표시 → 다음 활성화**  
* **건너뛰기: 선택 없이 다음 버튼 → 미응답 처리**

**S4. 이전 버튼**

* **해당 문제 초기화 상태로 이동**  
* **재응답 시 축 2(틀린 횟수) 누적 반영**  
* **축 1, 축 3 변동 없음**

**S5. 3축 통계**

* **축 1: 해당 Round에서 1번 이상 응답한 문항 수**  
* **축 2: OX\_ID별 틀린 횟수 — 전 라운드 영구 누적, 절대 리셋 없음**  
* **축 3: 전체 누적 응답 문항 수 — 라운드 완주마다 이월**

**S6. 완주 및 라운드**

* **150번째 문제 응답 시 완주**  
* **자동으로 Round \+1, 1번부터 재시작**  
* **건너뛴 문제 있어도 완주 처리**

**S7. 틀린 문제 모아풀기**

* **필터: 1회 이상 / 2회 이상 / 3회 이상 틀린 문제**  
* **축 2 기반, 동일 S2\~S4 플로우**

**S8. 이어하기**

* **과목별 마지막 위치 저장**  
* **홈 재진입 시 이어서 풀기 제공**
