## 버그 상세 분석

### BUG-1: T-08 — D-day 음수 시 "시험 종료" 미표시

**현상:** 시험일이 지난 경우 `D+N` 형태로 표시됨
**기대:** "시험 종료" 텍스트 표시

**원인 파일:** `DdayBanner.jsx`, `Home.jsx` 헤더 섹션

**수정 내용:**
```js
// DdayBanner.jsx — 변경 전
: `D+${Math.abs(dday)}`
// 색상: 'text-gray-500'

// DdayBanner.jsx — 변경 후
: '시험 종료'
// 색상: 'text-gray-400' (더 연한 회색으로 통일)

// Home.jsx 헤더 — 변경 전
`D+${Math.abs(dday)}일`

// Home.jsx 헤더 — 변경 후
'시험 종료'
```

---

### BUG-2: T-09 — 과목별 학습 탭 목록에 ⚠️ 누락

**현상:** Home.jsx 과목별 학습 탭의 세부과목 목록에서 40% 미만 시 빨간색은 표시되나 ⚠️ 없음
**기대:** 빨간색 + ⚠️ 표시 (SubjectStats.jsx와 동일)

**원인 파일:** `Home.jsx` (과목별 학습 탭 정답률 렌더링)

**수정 내용:**
```jsx
// 변경 전
<span className={`text-sm font-semibold w-10 text-right ${getAccuracyColor(pct)}`}>
  {pct !== null ? `${pct}%` : '-'}
</span>

// 변경 후
<span className={`text-sm font-semibold w-16 text-right shrink-0 ${getAccuracyColor(pct)}`}>
  {pct !== null ? `${pct}%${pct < 40 ? ' ⚠️' : ''}` : '-'}
</span>
// w-10(40px) → w-16(64px): ⚠️ 이모지 공간 확보
```
