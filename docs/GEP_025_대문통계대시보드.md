## 컴포넌트별 설계

### DdayBanner.jsx
| prop | 타입 | 설명 |
|------|------|------|
| `name` | string | 사용자 이름 (없으면 '오늘도 화이팅!') |
| `dday` | number\|null | 양수: D-N, 0: D-Day, 음수: D+N, null: 미표시 |

D-day 색상: 당일=빨강, 시험 전=파랑, 지남=회색

### ProgressStats.jsx
| 수치 | 계산식 |
|------|--------|
| 누적 완료 | `stats.total.solved` |
| 진도율 | `min(100, round(solved / 1080 * 100))`% |
| 정답률 | `round(correct / solved * 100)`% |

### TodayStats.jsx
| 수치 | 데이터 |
|------|--------|
| 문제 | `daily.solved` |
| 정답 | `daily.correct` |
| 정답률 | `round(correct / solved * 100)`% |

### SubjectStats.jsx — 12과목 구성
| 섹션 | 과목 (컬러) | 세부과목 |
|------|------------|---------|
| 법령 | 파랑 | 보험업법, 상법, 세제재무, 위험관리 |
| 손보1부 | 초록 | 보증보험, 연금저축, 자동차보험, 특종보험 |
| 손보2부 | 보라 | 재보험, 항공우주, 해상보험, 화재보험 |

**색상 규칙:**
- 60% 이상: `bg-green-500` (초록)
- 40~60%: `bg-yellow-400` (노랑)
- 40% 미만: `bg-red-500` + ⚠️ (빨강)
- 데이터 없음: 회색 바 + "-"

**접이식 동작:** 각 섹션(법령/손보1부/손보2부) 독립 접기/펼치기, 초기 상태 = 접힘

## Home.jsx 추가 내용

### 추가된 import
```js
import useStatsStore from '../stores/statsStore'
import DdayBanner from '../components/DdayBanner'
import ProgressStats from '../components/ProgressStats'
import TodayStats from '../components/TodayStats'
import SubjectStats from '../components/SubjectStats'
```

### 추가된 선언
```js
const stats = useStatsStore((s) => s.stats)
const today = new Date().toISOString().slice(0, 10)
const todayStats = stats.daily[today] ?? { solved: 0, correct: 0 }
```

### 추가된 JSX (기존 헤더 아래, 회차 선택 위)
```jsx
<section className="bg-gray-50 rounded-xl p-4 flex flex-col gap-4 border border-gray-100">
  <DdayBanner name={settings.name} dday={dday} />
  <div className="border-t border-gray-200" />
  <ProgressStats total={stats.total} />
  <TodayStats daily={todayStats} />
  <SubjectStats bySubject={stats.bySubject} />
</section>
```
